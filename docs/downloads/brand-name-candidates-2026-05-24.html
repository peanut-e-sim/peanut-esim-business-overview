Verification methodology

⚖️ USPTO TM Class 9 — Software, mobile apps, electronics. Searched via Justia Trademarks (static HTML index of live USPTO filings).
📡 USPTO TM Class 38 — Telecommunications services. Same Justia search; flagged any live registration or pending application.
🌐 .com domain — WebSearch + direct lookup. "Taken" = registered by any party. "Parked" = registered but no active site. "Available" = unregistered.
📸 @instagram handle — WebSearch inference (Instagram blocks direct scraping). ✓ = clearly available or no results; ✗ = active profile confirmed.
🎵 @tiktok handle — WebSearch inference. Same rules as Instagram.
📱 App Store conflict — WebSearch for "[name] app" to check if a live app already uses the mark in a conflicting category.

Verdict legend

GREEN — All checks clear. Safe to file.
YELLOW — One minor issue (expensive domain, dormant handle, unrelated TM). Proceed with legal counsel.
RED — Hard conflict: live Class 9/38 TM, major brand name collision, or domain owned by a competing product.
