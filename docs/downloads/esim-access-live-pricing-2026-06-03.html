The headline

eSIM Access is Tier 2.5 — one rung above Samurai (Tier 3), one rung below Flit's Korean MVNO (Tier 2). Real wholesale prices come in dramatically below the Peanut $2.26/GB Tier-3 assumption: UAE $1.28-1.50/GB (~33% cheaper than Samurai), Japan/Singapore/UK $0.34/GB on bulk plans (~85% cheaper), USA $0.50-0.66/GB. The Walnut Operator-advisor model assumed $1.34/GB blended Tier-2 — this is essentially confirmed for UAE and far exceeded everywhere else.

Where eSIM Access sits in the supply chain

What this changes about supplier strategy

What stays

  • Aggregator-direct is still the right path — exactly what the Walnut council recommended.
  • Tier-2.5 cost layer beats Tier-3 by ~33% on UAE and ~70-85% everywhere else.
  • API surface is complete for the planned Peanut feature set: ordering, top-up, refund/cancel, webhooks.

What changes

  • The Walnut "3-6 month Tier-2 vetting" timeline is moot. eSIM Access reseller account already exists and is API-ready. Integration can begin this week.
  • Samurai negotiation is no longer urgent. Going Samurai = paying eSIM Access's prices + 40% markup for the same underlying supply.
  • Flit's structural cost moat narrows. Flit only beats eSIM Access in Japan/Korea/China (direct bilateral countries). Everywhere else — including Peanut's home UAE market — both sit on the same Tier 2.5 layer.
  • Operator advisor's "Tier-3 net contribution goes negative" warning is resolved by going aggregator-direct, before any pricing changes.

What's still open

  • Pre-pay only model — needs working capital sitting in the account at all times (sized to ~30 days of forecast usage)
  • USIP vs non-USIP plan variants — pick which is right for your activation flow before catalog launch
  • Not all plans support top-up — must filter by supportTopUpType before exposing to users
  • Compare eSIM Access against the other Tier 2.5 aggregators (eSIM Go, Maya Mobile, Sparkle) when their partner-team replies hit the inbox
